3D Printed Knurled Nut "Suction Cup" Driver Bit

This page is a write up with information on how to customize, print, and use this 3D printed knurled nut driver bit. This bit is unlike others in that it is designed to be used with a power tool. It does not have any risk of "over-tightening" or "over-torque-ing" the nuts and destroying a jack. It is designed to slip rather than "over-torque".




Other People's Designs

Traditionally, most designs rely on two little "slots" or "notches". The tools slide into these notches, the same way a screwdriver slides into a horizontal ("flathead") or cross ("Phillips") slot. Sometimes it is hard to line up with the "slots" on a 3.5mm knurled nut, so some tools also provide an alignment pin which goes into the 3.5mm jack itself.
You can find a list of alternative tools at the bottom of the README of the GitHub repository.

The Design

Initially, I designed a bit to also fit in these "slots". However, these quickly wore out. Instead, this bit is designed to use the force of friction.
